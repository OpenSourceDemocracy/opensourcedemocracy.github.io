About

Open Source Democracy is a movement to get citizens involved in creating open source projects that improve democracy. The main goal of the organization is to encourage and support new members. We want to help new users learn how to use git and other tools to work together on large projects. Joining a new project or starting a new project can be daunting, especially for new programmers. We want to create a welcoming place which seeks to help new people achieve their goals and find a place to help foster their ideas. Most open source communities are focused a single product written in a particular programming language or framework. It takes a considerable amount of time to read and understand the code. Then even more time to find a change and test that it works. This proposed change then must be scrutinized by the more hardened users.. OSD wants to be the umbrella with helps to organize and collaborate across many different projects of varying complexity and domains, but all with the goal of improving democracy. Why do people help write and maintain open source software? Maybe they want a product or want to extend it, but for most it is a portfolio they can use on their Resume. Starting a project such as an App or website can be very daunting. First there are a lot of decisions for what framework or API you want to use. Then you must understand how to structure your project.
